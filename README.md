# MoneyStatBot
Telegram-бот для учета финансов. Он может сохранять денежные расходы и доходы, показывать историю операций и статистику за разные периоды.
### Стек технологий
* Java 22
* Spring Boot
* Spring MVC
* Spring Data
* PostgreSQL
* Lombok
* Telegram API
